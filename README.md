# Task Manager API 
Task Manager API is a **RESTful backend application** built with **Spring Boot**. It allows managing **tasks** and **users**, providing a clean architecture, proper separation of concerns, and a complete set of integration and unit tests.

## Features
- Manage **tasks** (CRUD operations)
- Manage **users** (CRUD operations)
- Built with **RESTful architecture**
- Fully tested with **unit** and **integration tests**

## Tech Stack
- Java 17
- Spring Boot
- JPA / Hibernate
- H2 Database (for testing)
- JUnit & MockMvc

## How to Run
```bash
./mvnw spring-boot:run
